== Bootstrap

[source,bash]
----
git clone https://github.com/<org>/ike-komet-ws.git
cd ike-komet-ws
mvn ws:init         # <1>
mvn clean install   # <2>
----
<1> Clones all subproject repos in dependency order; installs Maven
    wrapper and JVM config per subproject.
<2> Builds the full stack.

== Workspace Commands

All `ws:` goals appear in the IntelliJ Maven tool window
(under _Plugins > ws_). Double-click any goal to run it.

[source,bash]
----
mvn ws:overview          # Workspace overview
mvn ws:add -Drepo=      # Add a subproject repo
mvn ws:scaffold-upgrade-draft   # Preview scaffold upgrades
----
